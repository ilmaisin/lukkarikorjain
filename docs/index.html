lukkari.iirolaiho.net

This website used to host a script that fixed the iCalendar output of the Mimosa scheduling software used at Turku University of Applied Sciences. This service is discontinued because it is not useful with the new scheduling software Peppi.

As of August 2018, the new scheduling software cannot output iCalendar feeds, but the functionality should eventually be available (source, needs login).

The source code of the script is available at the Github page.
